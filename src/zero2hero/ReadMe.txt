Notebooks for base R
Versión modificada de las notebooks de la materia Laboratorio de Implementación I de la Maestria en Data Mining Universidad Austral

Las notebooks se ejecutaron desde Jupyter Lab utilizando un kernel de R bajo un environment creado para la materia.
Pasos para instalar R y disponer de un kernel:
Crear un environment para la materia
Activar el environment
conda install -c r r-irkernel 
run jupyter lab
Si hay problemas de compatibilidad o de instalación se recomienda instalar los paquetes desde el prompt usando: conda install -c r r-essentials
